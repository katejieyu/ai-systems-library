# AI Systems Library

A growing collection of reusable AI systems designed to structure product, design, and decision-making workflows.

Instead of relying on one-off prompts, this repository organizes AI capabilities into reusable systems with:
- defined workflows
- modular skills
- execution rules
- consistent output structures

## Systems

### Product AI System
Supports idea → problem definition → solution generation → challenge → refinement → execution readiness.

### Portfolio AI System
Supports portfolio audit → hiring simulation → UX QA → rewrite → execution planning.

## Why this repository exists

Most AI usage is fragmented and session-based.
This library treats AI as reusable workflow infrastructure instead of ad-hoc prompting.
