# Skill -- Portfolio Consistency

## Purpose

Learn the existing portfolio as a **coherent system**, not isolated pages.

This skill extracts:
- positioning patterns
- tone and writing style
- structural conventions
- CTA and labeling patterns

It ensures that any new or updated content:
- stays consistent
- reinforces the same level (Staff / Principal)
- does not dilute the overall signal

---

## When to Use

Use this skill when:

- adding a new page
- creating a new case study
- rewriting homepage or sections
- checking cross-page consistency
- aligning tone across multiple pages

Do NOT use when:
- building a portfolio from scratch (no existing system yet)

---

## Instructions

### Step 1 -- Read as a System (NOT page-by-page)

Analyze all provided pages together.

Identify:
- repeated structures
- recurring phrasing
- consistent vs inconsistent elements

---

### Step 2 -- Extract Core Positioning

Determine:

- What role is this portfolio signaling? (Senior / Staff / Principal)
- What are the dominant themes? (AI, systems, product, execution)
- What keywords define the positioning?

---

### Step 3 -- Extract Tone & Writing Style

Identify:

- sentence style (short vs long, direct vs abstract)
- level of abstraction
- use of technical vs conceptual language
- how claims are framed (impact vs philosophy)

---

### Step 4 -- Extract Structural Patterns

Identify:

- section structure (eyebrow -> title -> description -> CTA)
- card patterns
- hierarchy conventions
- naming consistency (sections, labels)

---

### Step 5 -- Extract CTA Patterns

Identify:

- how CTAs are written
- tone (action vs vague)
- consistency across pages

---

### Step 6 -- Identify Inconsistencies

Call out:

- tone mismatches
- repeated but inconsistent phrasing
- sections that break structure
- pages that feel like a different "voice"
- weak or diluted signals

---

### Step 7 -- Define Consistency Rules

Create a reusable system:

- positioning rules
- tone rules
- structure rules
- CTA rules
- "always include" signals
- "avoid" patterns
